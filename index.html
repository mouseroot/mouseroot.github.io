Mouseroot who?

Just your avarage self-taught game designer and computer programmer, I love code that is fast and effecient, I hold a firm beleif that we should have amazing technology, However we are a bit blind to the cost of making things easier, the code is easier to write, but the end result is a clunky mess

I have always loved the idea of linux and how we can have full control over all the processes and functions that happen within our own computer

Always passionate about developing smaller and most robust software, and very passionate about the golden age of gaming and how limiting situations of size led to amazing innovations of game design we still use today
